Условие: Монету подбрасывают три раза. Какова вероятность того, что выпадет ровно два орла?
Ожидаемый результат: Описание решения и ответ

Решение:

У монеты 2 стороны и вероятность выпадения одной из сторон равна 0,5
Так как ее подбрасывают 3 раза, то для вычисления вероятности получения определенного результата(любого) нужно перемножить 0,5 на саму себя 3 раза
0,5*0,5,*0,5= 0,125
Ответ: вероятность выпадения 2 орлов = 0,125
